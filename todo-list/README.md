# To-Do List

Ruby on Rails application to manage tasks.


## Running the app


Install all dependancies:

```
$ bundle install
```

Create the database and run migrations:

```
$ rake db:create db:migrate
```

Start the server:

```
$ rails server
```

And open the app in your browser at <http://localhost:3000/>.

## Testing the app

Run the tests:

```
$ rake
```
